# Security Policy

## Scope

This is a GitHub profile repository. It does not handle user data, credentials, or production systems.

## Reporting Issues

If you find a vulnerability in a referenced project (Proxy Protocol, Developer Tools, etc.), please report it directly to the relevant repository.

For issues specific to this profile repository (e.g., a malicious script embedded in an SVG, a broken workflow leaking secrets):

- **Email:** mail@xconmax245.dev
- **Subject:** `[SECURITY] Profile Repository`

Do not open a public GitHub issue for security concerns.

## Response Timeline

- Acknowledgement within 48 hours
- Resolution or mitigation within 7 days for confirmed issues

## Supported Versions

Only the latest commit on `main` is supported.
